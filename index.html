Calendar-app

The calendar-app is a graphical user interface for social calendar application. This graphical user interface will support will support a web, mobile web and mobile devices. Primary technologies used will be: HTML5, CSS3, AngularJS, AngularJS UI, JQuery, JQuery UI, and JQuery Mobile.
